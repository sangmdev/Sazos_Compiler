SAZO'S Compiler

The Mission

Why use our application?

When we set out to build this application we envisioned something that would make coding in assembly language easier, simpler and more interactive for CS2700 UMSL students. Our team is made up of UMSL students who have completed CS2700.

Features

File system user interface

There is a file system user interface to avoid use of terminal and allows for upload and download.

Download the program

Download the program as a .txt or .asm file in order to submit to instructor.

Color coded commands

The platform has color coded commands for easy usability.

Error Checking

The platform has a robust error checking so you catch all errors as you code.

Tutorial & Documentation

There is a comprehensive technical document on the assembly language instruction set so that you can reference at any time and a Tutorial page that you can use to learn how to use our compiler
